Council Launches Event Toolkit to Support Organisers

Dec 15, 2023 | News | 0

The Daylesford and District Pipe and Drum band led the Highland Gathering parade.

Hepburn Shire Council has launched a new event toolkit to help local event organisers to plan and carry out events successfully.

Mayor Cr Brian Hood said whether the event was big or small, the Event Toolkit was a one-stop-shop for event organisers.

“Events are an important drawcard for community and visitors in our Shire and a driver in our local economy. The toolkit features information on event planning, risk management, sponsorship, grants, coordinating volunteers and staff, how to run committee meetings and post-event reviews and surveys,” said Cr Hood.

“It aims to make it easy to access everything organisers need when planning an event in our Shire,” he said.

The development of the toolkit delivers on two aspects in Council’s Event Strategy 2020-2025: to build the capacity of the event industry and to enhance and measure the value and impacts on events.

“The toolkit provides event organisers with the resources, templates and tools to run events that are successful, sustainable and compliant,” said Cr Hood.

The resources are comprehensive and an important tool for community groups and event organisers to help guide them with the event planning process. The toolkit caters for events of all sizes and all types and will be updated as information changes.

The Event Toolkit is available on Council’s website at www.hepburn.vic.gov.au/Event-toolkit

This article is based on a media release from Hepburn Shire Council.
